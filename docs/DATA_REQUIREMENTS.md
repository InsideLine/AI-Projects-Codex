# Data Requirements

## SOLO Licensing Software

Required fields:

- Activation Date
- License ID
- Company Name
- License Entered Date
- Status
- IP Address
- Initial Product Version
- Deactivated Date

Useful additions:

- Activation ID or event ID
- Product SKU
- License type
- User or contact tied to activation
- Any machine fingerprint available from SOLO

## AWS Customer Usage Information

Required fields:

- License ID
- Links Processed
- Files Processed
- File Size in Bytes
- Process Name
- Start Time
- End Time
- Company Name
- Machine Name
- Username
- MAC address
- IP Address, when available
- Tenant Name
- Site Name
- Tenant ID or equivalent
- Database Name

Useful additions:

- Source AWS account ID
- Region
- Product version
- Build number
- Job/run ID
- Record creation timestamp

## Zoho CRM

Required fields:

- Company/account ID
- Canonical company name
- License ID or mapping to licenses
- Personnel licensed
- Organization definition text
- Covered countries, states, cities, or sites if structured
- Contract/EULA version
- License start and end dates
- Account owner or legal contact

Useful additions:

- Parent/subsidiary relationships
- Known aliases
- Customer domains
- Contract amendments
- Renewal history

## Geolocation

Required fields:

- IP address
- City
- State or region
- Country
- Provider
- Lookup timestamp

Strongly recommended fields:

- Accuracy radius in kilometers
- Latitude and longitude
- ISP/organization
- ASN
- Connection type
- VPN/proxy indicator, if licensed
